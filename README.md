# Simple Calculator with History in C

A console-based calculator program built using C that performs basic arithmetic and stores the last 5 calculations.

## Features
- Addition, Subtraction, Multiplication, Division
- Stores last 5 calculations
- Menu-driven interface
- Exit option

## How to Run
1. Compile the program:

    gcc calculator_history.c -o calculator_history

2. Run:

    ./calculator_history
